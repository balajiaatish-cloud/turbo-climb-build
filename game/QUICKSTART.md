# Quick Start

1. Open `index.html`.
2. On phones, rotate to landscape for the cleanest layout and touch controls.
3. Click `GARAGE` if you want to buy upgrades or unlock vehicles.
4. Click `MAPS` if you want to unlock Greenland or Moon for 10 coins.
5. Click `REWARDS` if you want to claim the daily login or 30-minute playtime bonuses.
6. Click `GOALS` if you want to track missions and achievements.
7. Click `START`.
8. Climb as high as you can without dropping 30 m below your best height.
9. Watch the run get harder as you climb higher.
10. Use the install button in supported browsers if you want it as an app.

## Controls

- `A` / `Left Arrow`: move left
- `D` / `Right Arrow`: move right
- `W` / `Up Arrow` / `Space`: boost jump
- Mobile: use the on-screen touch buttons

## Tips

- Save fuel for big gaps.
- Fuel canisters and repair kits are different pickups now.
- Diamonds are rarer and worth grabbing whenever the path is safe.
- Hazards hit hard, so recover before forcing another jump.
- Upgrades and owned vehicles stay saved between runs.
- Higher zones use smaller landings, bigger gaps, and denser hazards.
- Moon uses lower gravity, so your jumps stay in the air longer.
- Daily rewards double their coin payout on consecutive claim days.
- Height rewards start at 200 m and double every extra 100 m.
- Each map now has its own hazard style and knockback feel.
- Chrome, Edge, and Android browsers can install the game like an app.
